--- Consultas Relacionais no SQL Server
O objetivo é aplicar os conhecimentos de SQL em um banco de dados relacional que simula um site de filmes, realizando 12 consultas para extrair informações relevantes.

--- Estrutura do Banco de Dados
-- O banco é composto pelas seguintes tabelas:
- Filmes: informações sobre os filmes (nome, ano, duração)
- Atores: dados dos atores (nome, sobrenome, gênero)
- Generos: lista de gêneros disponíveis
- ElencoFilme: relacionamento entre filmes e atores, incluindo o papel de cada ator
- FilmesGenero: relacionamento entre filmes e seus gêneros

--- Consultas realizadas
- Buscar nome e ano dos filmes
- Buscar nome e ano dos filmes ordenados por ano crescente
- Buscar o filme "De Volta para o Futuro" com nome, ano e duração
- Buscar filmes lançados em 1997
- Buscar filmes lançados após o ano 2000
- Buscar filmes com duração entre 100 e 150 minutos, ordenados por duração
- Buscar quantidade de filmes por ano, ordenado pela quantidade
- Buscar atores do gênero masculino
- Buscar atores do gênero feminino, ordenados pelo primeiro nome
- Buscar nome do filme e seu(s) gênero(s)
- Buscar filmes do gênero "Mistério"
- Buscar nome do filme e os atores com seus papéis
